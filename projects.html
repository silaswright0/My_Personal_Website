Mancala Game

This Java program supplies a GUI and class hierarchy with the ability to play the popular board game Mancala with different rule sets. The code also provides the user with the ability to save and load game states, as well as access information about a user's personal profile. I used JUnit testing and PMD error checking to ensure professional code quality. This assignment was for my CIS*2430 class which teaches Java and Object Oriented Programming.

Data Structures Implementations

For my CIS*2520 course, I was responsible for providing abstract implementations of several data structures including linked lists, hash tables, and tries. Each implementation required methods for insertion, query, deletion, and as such detailed knowledge of the data structure. Each implementation was thoroughly tested with various sizes and styles of data highlighting the strengths and weaknesses of each data structure.
